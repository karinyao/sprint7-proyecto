# sprint7-proyecto
Descripción del Proyecto

Esta aplicación web interactiva fue desarrollada como parte del Sprint 7 del bootcamp de Data Analyst en TripleTen. El objetivo principal es proporcionar una herramienta de visualización de datos que permita explorar y analizar un conjunto de datos de anuncios de venta de coches en Estados Unidos.

¿Para qué sirve esta aplicación?
La aplicación permite a los usuarios explorar visualmente la relación entre diferentes características de los vehículos, específicamente:

* Distribución del kilometraje: Visualizar cómo se distribuyen los kilometrajes de los vehículos anunciados
* Relación precio-kilometraje: Analizar cómo el kilometraje de un vehículo afecta su precio de venta

Funcionalidad
La aplicación proporciona las siguientes características interactivas:

1. Información del Dataset

* Muestra el número total de anuncios en el conjunto de datos
* Proporciona contexto sobre los datos que se están analizando

2. Histograma de Kilometraje
* Activación: Mediante casilla de verificación
* Función: Visualiza la distribución de kilometraje de todos los vehículos
* Utilidad: Permite identificar los rangos de kilometraje más comunes en el mercado

3. Gráfico de Dispersión (Kilometraje vs Precio)
* Activación: Mediante casilla de verificación
* Función: Muestra la relación entre el kilometraje y el precio de los vehículos
* Utilidad: Ayuda a entender cómo el uso del vehículo (kilometraje) influye en su valor de mercado

Tecnologías Utilizadas
* Python 3.x: Lenguaje de programación principal
* Pandas: Librería para manipulación y análisis de datos
* Plotly Express: Librería para crear visualizaciones interactivas
* Streamlit: Framework para desarrollar la aplicación web
